# simple-maze-solver
A simple program that helps the player solve the maze.

### Sample maze (.txt file)
The .txt file consists:
* the size of the maze
* the entrance or the starting point
* and the maze itself
  * 0 = wall
  * 1 = path
  * 2 = destination
